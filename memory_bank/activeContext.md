## Ключевая связка (из изучения braine)
Наш ETL 1С (холодный контур плана) должен писать выгрузки как md-файлы + md-таблицы в KB-репо → braine индексирует их и отвечает с цитатами, агрегаты считает SQL-скилл. Braine закрывает «горячий контур» и бот-интерфейс целиком; наша работа — ETL 1С → KB-репо + golden set под 1С-домен.
