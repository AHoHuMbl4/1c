# Product Context

## Зачем существует проект
Дать бизнесу «второй мозг» по данным его 1С: сотрудник спрашивает в Telegram —
бот отвечает по актуальным данным компании (контрагенты, продажи, склад, деньги…)
с цитатами на источники, строго на чтение (в 1С ничего не меняется).

## Что получилось (работает)
- Штатный OData 1С (read-only) → внешняя база знаний → RAG-бот с цитатами.
- Раскатка на новый бизнес = разовая настройка (доступы + галочки «что тянуть») →
  дальше автоматически (ночная выгрузка, индексация, ответы). Zero-touch, переживает ребут.
- Универсальный код + конфиг под бизнес — копипаст на любую конфигурацию 1С
  (Бухгалтерия / ERP / УТ …).

## Ценность
- Приватность/безопасность: 1С только читаем, данные под контролем.
- Точность: бот не выдумывает (гейты braine — каждая цифра сверяется, обязательные цитаты);
  нет данных → честное «нет данных».
- Историчность: KB накапливает снапшоты («как менялось»), 1С хранит «сейчас».

## Кто пользователи
- Владелец (Евгений) и сотрудники бизнеса, которым нужны ответы по данным 1С.
- На проде — клиенты (каждый бизнес — своя настройка).
