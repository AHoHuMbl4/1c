# Product Context

## Зачем существует проект
Дать AI-боту доступ к данным 1С:ERP (локальный сервер) через MCP-протокол: произвольные запросы на чтение, вычисления на отдельном сервере. Решение должно быть бесплатным (open source).

## Что должно получиться
- Работающий MCP-сервер, подключённый к 1С:ERP на Windows-машине.
- AI-клиент (Claude и др.) выполняет произвольные запросы к данным 1С через инструменты MCP (`execute_query`, `get_metadata`, …).
- Развёрнуто: MCP-часть на LXC `lxc-claude-1c` (Ubuntu), 1С-часть на Windows `10.8.0.58`.

## Кто пользователи
- Владелец (Евгений) и его AI-боты/агенты.
