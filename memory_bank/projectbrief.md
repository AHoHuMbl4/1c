# Project Brief — «Второй мозг» на данных 1С

## Суть проекта
AI-бот отвечает на вопросы по данным компании из 1С — строго на чтение. Данные 1С
регулярно выгружаются во внешнюю базу знаний, индексируются, бот отвечает с цитатами.
Работает end-to-end; раскатывается копипастом на любую конфигурацию 1С с разовой
настройкой под бизнес.

**Полная картина — `docs/ARCHITECTURE.md`. Раскатка — `docs/RUNBOOK_DEPLOY.md`.**

## Как (кратко)
Штатный OData 1С на IIS (read-only, пользователь `ai_reader`) → LXC: GET-only шлюз →
ETL выгружает выбранные сущности в KB-репо (GitLab) → braine (Open WebUI + pgvector)
индексирует → Telegram-бот отвечает. «Что тянуть» выбирается галочками (config-ui).

## Инвариант
🔴 1С только читаем. Ни один запрос не меняет данные 1С (два слоя: `ai_reader` + GET-only шлюз).

## Границы
- Работа только в папке `1c/` (этот репозиторий).
- Каждое изменение = отдельный git-коммит + push в GitHub `origin/main`.
- На серверах (LXC/Windows) состояние меняем только по явной инструкции владельца.
