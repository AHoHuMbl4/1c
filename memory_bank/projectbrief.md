# Project Brief — 1C

## Суть проекта
Подключение AI-бота к 1С:ERP через MCP-сервер: тестируем бесплатные open-source MCP-решения для произвольных запросов к данным 1С (чтение), вычисления — на отдельном сервере. Ресёрч-основа: `docs/research/1c-mcp-server-research.md`.

Кандидаты на тест (по ресёрчу): 1C MCP Toolkit (ROCTUP) — основной, 1c_mcp (Владимир Харин), mcp-1c (feenlace), 1c-rest-mcp, http1c.

## Границы
- Вся работа ведётся **только** в папке `1c/` (этот репозиторий). Смотреть в другие папки на машине запрещено.
- Каждое изменение фиксируется отдельным git-коммитом и пушится в GitHub.

## Ключевые ссылки
- Remote: https://github.com/AHoHuMbl4/1c.git (ветка `main`)
- Memory bank: `memory_bank/` — читать в начале каждой сессии, обновлять при каждом значимом изменении.
