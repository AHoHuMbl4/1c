## Окружение разработки
- Рабочая папка: `/srv/data/cursor/cursor/1/1c` (Linux-машина владельца).
- Развёртывание — на удалённом сервере (создаёт владелец, 2026-07-22). Адрес/пользователь — уточнить, когда сервер будет готов.

## Доступ к серверу
- SSH-ключ Claude (dedicated, только для этого проекта): приватный `~/.ssh/id_ed25519_1c` на машине владельца, публичный добавляется на сервер.
- Подключение: `ssh -i ~/.ssh/id_ed25519_1c <user>@<host>`.
