# 1C

Рабочий репозиторий проекта по 1С:Предприятие.

## Правила
- Вся работа ведётся только внутри этой папки.
- Каждое изменение — отдельный git-коммит с пушем в `origin/main`.
- Контекст проекта живёт в [`memory_bank/`](memory_bank/) — читать в начале сессии, обновлять по ходу работы:
  - `projectbrief.md` — суть и границы проекта
  - `productContext.md` — зачем и для кого
  - `techContext.md` — платформа, инструменты, окружение
  - `systemPatterns.md` — архитектура и соглашения
  - `activeContext.md` — текущий фокус и следующие шаги
  - `progress.md` — что сделано / в работе / бэклог
